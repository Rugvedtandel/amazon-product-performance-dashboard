# Amazon Product Performance Dashboard

## Project Overview

This project analyzes Amazon product data using Python, SQL, and Power BI to uncover insights related to product ratings, reviews, pricing, discounts, and category performance.

## Tools Used

- Excel
- MySQL
- Python (Pandas, NumPy)
- Power BI
- DAX

## Dataset

- Total Products: 1,465
- Categories: Multiple Amazon product categories
- Metrics:
  - Product Rating
  - Review Count
  - Actual Price
  - Discounted Price
  - Discount Percentage

## Dashboard KPIs

- Total Products: 1,465
- Average Rating: 4.1
- Total Reviews: 27 Million
- Average Discount: 47.44%

## Key Insights

- Electronics is the largest product category.
- Office Products has the highest average rating.
- Highly rated products tend to receive more reviews.
- Discounts average nearly 47%, indicating aggressive promotional pricing.
- A small number of products contribute to a significant share of total reviews.
